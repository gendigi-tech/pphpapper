# HHP Global – Company Profile Landing Page

A clean, branded landing page for HHP Global JSC's company profile.

## Features
- 📄 Embedded PDF viewer (Google Drive)
- ⬇️ One-click PDF download
- 🎨 On-brand blue & black design
- 📱 Fully responsive

## Deploy to Vercel
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Fork or import this repo into Vercel
2. No build step needed — static HTML
3. Set root directory to `/`

## Tech
- Pure HTML/CSS/JS — zero dependencies
- Self-contained (logos embedded as base64)
- Fonts via Google Fonts CDN
